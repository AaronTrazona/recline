Uploading

You can bulk upload an array of JSON documents to a Couch via HTTP POST. Make sure that they are formatted one document per line:

  {
    "docs": [
      {"name": "barry", "type": "human"},
      {"name": "carl",  "type": "human"},
      {"name": "ethel", "type": "robot"}
    ]
  }
